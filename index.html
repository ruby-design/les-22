Welkom website van Ruby Design!

Dit is mijn eerste Bootstrap-website. Bootstrap is een gratis open source. Een verzameling van hulpmiddelen voor het maken van websites. Het systeem bevat verschillende ontwerpsjablonen (HTML, CSS en JavaScript) en interfacecomponenten die voorgedefinieerd zijn. Voor deze website heb ik gebruik gemaakt van een aantal van deze componenten.

Amsterdam
